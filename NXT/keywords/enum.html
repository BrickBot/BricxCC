An enumeration is a user-defined type consisting of a set of named constants called enumerators. 

   enum Suit {
      Diamonds,
      Hearts,
      Clubs,
      Spades
   };

By default, the first enumerator has a value of 0, and each successive enumerator is one larger than the value of the previous one, unless you explicitly specify a value for a particular enumerator. Enumerators needn't have unique values within an enumeration. The name of each enumerator is treated as a constant and must be unique within the scope where the enum is defined.
