FindWorld(v, relationSource, criteria, threshold)Function - Spy

Sets variable v to the next entry in the world relationship table that matches the criteria specified.

task main()
{
  int v = -1;
  ClearWorld();

  while (v != -1)
  {

    SetTargetID(v);

  }
