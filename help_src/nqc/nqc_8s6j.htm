EVENT_MASK(event)Value – RCX2, Scout, Spy

Convert an event number to a mask. The monitor statement and some API functions (such as ActiveEvents() or Event()) need to handle multiple events. This is done by converting each event number to an event mask, and then combining the masks with a bitwise OR. For example, to monitor events 2 and 3, the

following statement could be used:

monitor(EVENT_MASK(2) | EVENT_MASK(3))
