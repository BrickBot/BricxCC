Clear Memory

Press the Clear Memory menu item in the Tools menu to clear the memory in the brick. This command removes all tasks and subroutines in all programs, and removes the RCX datalog (if any). Clearing memory is important when loading large programs because the brick keeps all programs and tasks; also if you switch it off. After clearing memory you have about 6K for your program in the RCX (using the RIS 1.0 and 1.5 firmware). For the CyberMaster this is about 512 bytes. The Scout has NNN bytes. The Spybot has 3.75K bytes.
