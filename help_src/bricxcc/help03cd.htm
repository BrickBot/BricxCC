Window List

Use this resizable dialog to quickly switch to a specific editor window. It is accessed via View|Window list (Alt-V, W) or via the hot key Alt-0 (zero).

Select the editor window you wish to activate in the list box and choose OK to close the list and activate the specified window. You can also double-click the desired item in the list or selected it and press the Enter key to perform the same action. A numeric list of editor windows is also maintained on the Windows menu.
