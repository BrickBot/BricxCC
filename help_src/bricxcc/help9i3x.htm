Select Series

Use this dialog to select one of the series in your chart and configure its settings. It is accessed from the Data Analysis dialog via the Select Series toolbar button (Ctrl-S) or by holding down the Ctrl key while clicking on the chart using the left mouse button. If you click directly on one of the series in the chart this dialog will be bypassed and you will go directly to the Series Configuration dialog.

Select the desired series using the list box. The series can then be chosen either via selecting the OK button or by double-clicking on the series name in the list box.
