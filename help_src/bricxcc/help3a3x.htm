Set Values

Use this dialog to set any writeable source/value combination to the value of any readable source/value combination. It is accessed via Tools|Set Values (Alt-T, L).

Some sources are writeable only when the value is within a narrower range than the range of valid source values. For example, the random source is only writeable when the value is set to zero. The list of sources and value ranges will depend on the brick you have configured BricxCC to communicate with.
