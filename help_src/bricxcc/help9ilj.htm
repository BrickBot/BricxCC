Editing a File

The editor windows support all the usual edit functions. You can type in text, select it, delete it, etc. The editor syntax highlights the text such that it is easy to distinguish different aspects of the program. Syntax highlighting can be switched off in the Preferences item in the Edit menu.

In the Edit menu you also find the usual cut, copy, and paste commands that are common in all editors, with the usual shortcuts. Last changes can be undone or redone.

As mentioned previously, the very important Preferences dialog is accessible via the Edit menu.

Users can also choose the Next Field option from this menu, which moves the cursor to the next block of text delimited by double quotes (e.g., "MY_TEXT"). This feature is more fully described in the Templates section.
