Not Quite C

Not Quite C (NQC) is an easy to use language for programming the Lego robots. The preprocessor and control structures of NQC are very similar to C. NQC is not a general purpose language - there are many restrictions that originate from actual limitations of the programmable brick.

NQC was written by Dave Baum, with recent enhancements by John Hansen. BricxCC simply calls the NQC compiler for compiling and downloading code to the brick. For more information on NQC look in the tutorial (which contains a reference guide) provided with BricxCC. For more information of NQC, see the web page

http://bricxcc.sourceforge.net/nqc/
