Find Brick

If the program did not find the brick at the startup, you can press the Find Brick menu item in the Tools menu to try again, e.g. after switching on the brick. If you don’t want to use the brick anymore, you can choose Turn Brick Off from the Tools menu. This will turn the robot off and close the communication. If you want to use the brick again later, turn it on and choose Find Brick.

You can also temporarily close the communication with the RCX, without turning the robot off, by choosing the Close Communication item in the Tools menu. This is useful if you want to run a different program that uses the COM port or USB tower, but don’t want to stop BricxCC. The communication is reopened the next time you select any BricxCC function that tries to communicate with the brick.
