Project Manager

Use this dialog to manage a list of files that are all part of the same project. It is accessed via View|Project Manager (Alt-V, P) or by its hotkey Ctrl-Alt-F11. This feature is only available when the active editor window contains a Pascal, C/C++, or Java source file.

A right-click popup menu provides access to the Add, Remove from Project, and Open features of this dialog. These can also be accessed via hotkeys. Use Ctrl-Ins, Ctrl-Del, and Ctrl-O respectively. You can also reorder the list of files by dragging and dropping a filename within the list box.
