Spybot EEPROM

Use this dialog to examine the Spybot EEPROM values. It is accessed via Tools|Spybot EEPROM (Alt-T, Y).

The Game Items tab contains two tabs. The first (Configuration data) contains a variety of controls which let you view or edit the motor control, species, long ID, user level, ping rate, runs, wins, losses, points, play seconds, total play time, max robots, mission ID, mission points, high score, robot name, birth date, robot data, and status values. The second tab (Missions & Tokens) allows editing the missions which have been run and the Spybot token values.

The Raw Data tab provides a byte-level view of the first 256 bytes of Spybot EEPROM. The first 128 bytes are editable via the Game Items tab and the data is shared between these two tabs. The remaining 128 bytes can only be edited and viewed via the Raw Data tab.

To read the Spybot EEPROM data use the Read button. This will populate the raw data fields on the second tab and update the controls on the Game Items tab. Once the data has been loaded you can save the data to a simple text file via the Save button. You can load previously saved Spybot EEPROM data using the Load button. After you edit the values you can write them to the Spybot using the Write button. Any attempt to write to the Spybot EEPROM requires that the user confirms the operation in order to avoid accidentally overwriting important data.

To view all the Spybot EEPROM data use the Custom tab. Here you can select the EEPROM block to start reading from and the number of blocks to read. Select the Read button to retrieve the specified block range. Select the Clear button to erase the contents of the data window. The selected contents of the data window can be copied to the clipboard via Ctrl-C or via the right-click popup menu.
